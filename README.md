# RunStack Open Source (runstack-ops)

Bem-vindo ao repositório open-source do RunStack! Aqui liberamos pacotes gratuitos, incluindo o Obfuscator PHP níveis 1-3 para proteção básica de código.

## Pacotes Disponíveis
- **Obfuscator**: Ofuscação PHP multi-nível (freemium).
  - Nível 1: Minificação (remove comentários/espaços).
  - Nível 2: Renomeação de variáveis.
  - Nível 3: Ofuscação de strings (base64/hex).
  - Para níveis avançados (4-6: criptografia AES, dead code, polymorphic), visite entre em contacto com o autor.

## Instalação
1. Clone o repo: 'git clone https://github.com/seu-usuario/runstack-ops.git'
2. Rode 'composer install'
3. Exemplo de uso: Veja 'packages/obfuscator/tests/test_level2.php'
